INTRODUCTION
------------

This module provides the 'advanced outsider permissions' to group members,
in addition to basic member permissions, so that an outsider does not lose
its rights when it becomes a member.

More info:

 * For a full description of the module, visit [the project page]
   (https://www.drupal.org/project/group_outsider_in).

REQUIREMENTS
------------

This module requires group modules outside of Drupal core.


INSTALLATION
------------

 * Install the Group outsider in module as you would normally install a 
   contributed Drupal module. Visit https://www.drupal.org/node/1897420 
   for further information.

CONFIGURATION
-------------

There is no configuration.
